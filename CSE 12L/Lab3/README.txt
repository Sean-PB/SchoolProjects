Lab 3: ASCII-risks (Asterisks)

-----------
DESCRIPTION

In this lab, the user enters a number. The number must be greater than 0 or the program will ask for 
input again. Once the input is greater than 0 the program makes a rectangle of the inputs height
with a series of ascending numbers and astricks.

----------
FILES

-
Lab3.asm

This file contains the program previously described.

----------
INSTRUCTIONS

This program is intended to be run using the MARS IDE. Open up Mars4_5 and load in the file.
Then you may run it and input a number for the height of the rectangle pattern.
