DESCRITION

The purpose of this lab was to develop skills in MML as well as to learn
about circuits and logic gates. This was a 3 part lab in which we used MML
to implement logic and truth tables by making circuits with various gates.

INSTRUCTIONS

Must have MML application. From there all you need to do is open the Lab1.lgi
file and that will bring you inside the application. Press the green play
button on the top task bar to start simulation. From there you can play with
the four switches below as you please.
