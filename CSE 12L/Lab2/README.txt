Lab 2: Simple Data Path

-----------
DESCRIPTION

In this lab the user inputs values and selects which register for them to be placed in.
Then, the user selects which register vaule to be used for the ALU inputs.
The ALU then takes those inputs and preforms a right rotate operation.
